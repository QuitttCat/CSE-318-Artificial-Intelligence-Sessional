## üß† Overview

This repository contains full implementations and reports for  
**CSE 318 ‚Äì Artificial Intelligence Sessional (BUET, Level 3 Term 1)**.

These assignments cover:

- Informed search (A*)
- Local search & metaheuristics (GRASP)
- Adversarial search (Minimax + Alpha-Beta)
- Machine learning (Decision Trees, Python + C++)

---

## Offline 1 ‚Äì A Star Algorithm (N-Puzzle)

- A* algorithm with multiple heuristics  
- Hamming, Manhattan, Euclidean, Linear Conflict  
- Optimal path generation  
- Visualization included  

---

## Offline 2 ‚Äì Local Search

- GRASP metaheuristic  
- Random, greedy & semi-greedy constructive phases  
- Local search improvement  
- Multiprocessing for speed  
- Benchmark graphs & CSV outputs  

---

## Offline 3 ‚Äì Adversarial Search

- Minimax with Alpha‚ÄìBeta pruning  
- Custom heuristic evaluation  
- State-space analysis  
- Works with interactive Chain Reaction (React UI)

## Offline 4 ‚Äì Decision Tree / Decision Tree C++ Implementation

- Tested on: Iris & Adult datasets  
- Overfitting, depth control & performance metrics

## ‚öô Installation & Running

### Clone Repository
```bash
git clone https://github.com/QuitttCat/CSE-318-Artificial-Intelligence-Sessional.git
cd CSE-318-Artificial-Intelligence-Sessional
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run Assignments
```bash
python final.py          # Offline 1
python GRASP.py          # Offline 2
python graph_generator.py
```

### Run Offline 3 (React UI)
```bash
cd assignment3
npm install
npm start
```
